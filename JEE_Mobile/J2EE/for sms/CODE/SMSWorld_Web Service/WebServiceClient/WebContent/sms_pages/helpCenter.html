Welcome To SMS WORLD...

If new user then follow the steps given below :

1. If you want to be a part of SMS World then for registering click on "Register here" button on login page

2. Fill the appropriate details in Registration Form and then click on Register.

3. After Registration your account will get created and your password will be sent to your mobile.
    Using your mobile number and password you can access your account directly.
    Now you can enjoy all the services provided through SMS World

4. The services are :

  1. Send SMS
    You can use the Send SMS facility to send the message.

  2. Save SMS
    You can use the Save SMS facility to save the message in draft.

  3. Address Book
    You can use the Address Book facility to keep the contacts of your friends in address book.

  4. Change Password
    You can use the Change Password facility to change your password.

  5. Template Messages
    You can use the Template Messages at the time of sending message.
